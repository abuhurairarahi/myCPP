1. #include<bits/stdc++.h> :  

        1.1. It includes almost all standard C++ libraries at once.

              1.1.1. #include <iostream>
              1.1.2. #include <vector>
              1.1.3. #include <algorithm>
              1.1.4. #include <map>
              1.1.5. #include <set>
              1.1.6. #include <string>

        1.2. Commonly used in competitive programming to save time.
        1.3. It is not part of the official C++ standard.
        1.4. It works mainly with the GNU Compiler Collection.
        1.5. Generally not recommended for production or professional code because it increases compile time.

    !Attention: "#include <bits/stdc++.h>" increases compile time because it includes many unnecessary standard library headers. It is mainly used in competitive programming with the GNU Compiler Collection.
